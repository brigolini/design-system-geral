Design Responsivo

A biblioteca @almg/native fornece um sistema de grid flexível e design tokens CSS que facilitam a criação de layouts responsivos e adaptáveis.

Sistema de Grid

O AlmgGrid cria layouts baseados em CSS Grid com colunas predefinidas. O AlmgGridItem controla quantas colunas cada item ocupa.

Colunas disponíveis

ValorUso típico
1Layout de coluna única (mobile-first)
2Duas colunas iguais (formulários simples, comparações)
3Três colunas (cards de resumo, dashboards)
8Grid de 8 colunas (layouts intermediários)
12Grid completo (layouts complexos com spans variáveis)

Exemplo: Grid 12 colunas com spans variáveis

Cada linha pode ter itens de tamanhos diferentes. Os spans devem somar o total de colunas:

import { AlmgGrid, AlmgGridItem, AlmgCard } from '@almg/native';

// Linha 1: sidebar (4 colunas) + conteúdo (8 colunas)
// Linha 2: duas metades (6 + 6)
// Linha 3: três terços (4 + 4 + 4)
<AlmgGrid columns={12} gap="md">
  <AlmgGridItem span={4}>
    <AlmgCard title="Sidebar">
      <p>Navegação ou filtros</p>
    </AlmgCard>
  </AlmgGridItem>
  <AlmgGridItem span={8}>
    <AlmgCard title="Conteúdo Principal">
      <p>Área de conteúdo</p>
    </AlmgCard>
  </AlmgGridItem>

  <AlmgGridItem span={6}>
    <AlmgCard title="Metade Esquerda">
      <p>Conteúdo</p>
    </AlmgCard>
  </AlmgGridItem>
  <AlmgGridItem span={6}>
    <AlmgCard title="Metade Direita">
      <p>Conteúdo</p>
    </AlmgCard>
  </AlmgGridItem>

  <AlmgGridItem span={4}>
    <AlmgCard title="Terço 1"><p>Info</p></AlmgCard>
  </AlmgGridItem>
  <AlmgGridItem span={4}>
    <AlmgCard title="Terço 2"><p>Info</p></AlmgCard>
  </AlmgGridItem>
  <AlmgGridItem span={4}>
    <AlmgCard title="Terço 3"><p>Info</p></AlmgCard>
  </AlmgGridItem>
</AlmgGrid>

Layouts Responsivos com Media Queries

O sistema de grid usa CSS puro. Para tornar o layout responsivo, combine com media queries no className ou em CSS separado:

Abordagem 1: CSS com classes utilitárias

Crie classes CSS no seu projeto que alterem o grid em breakpoints:

/* styles/responsive.css */
@media (max-width: 768px) {
  .grid-responsive {
    grid-template-columns: 1fr !important;
  }
}
<AlmgGrid columns={12} gap="md" className="grid-responsive">
  <AlmgGridItem span={4}>Sidebar</AlmgGridItem>
  <AlmgGridItem span={8}>Conteúdo</AlmgGridItem>
</AlmgGrid>

Em telas menores que 768px, ambos os itens ocuparão a largura total.

Abordagem 2: Grids diferentes por breakpoint

Use dois grids com exibição condicional:

import { AlmgGrid, AlmgGridItem, AlmgCard } from '@almg/native';

function Dashboard() {
  return (
    <>
      {/* Desktop: 3 colunas */}
      <div className="hidden-mobile">
        <AlmgGrid columns={3} gap="md">
          <AlmgGridItem><AlmgCard title="Card 1">...</AlmgCard></AlmgGridItem>
          <AlmgGridItem><AlmgCard title="Card 2">...</AlmgCard></AlmgGridItem>
          <AlmgGridItem><AlmgCard title="Card 3">...</AlmgCard></AlmgGridItem>
        </AlmgGrid>
      </div>

      {/* Mobile: 1 coluna */}
      <div className="hidden-desktop">
        <AlmgGrid columns={1} gap="sm">
          <AlmgGridItem><AlmgCard title="Card 1">...</AlmgCard></AlmgGridItem>
          <AlmgGridItem><AlmgCard title="Card 2">...</AlmgCard></AlmgGridItem>
          <AlmgGridItem><AlmgCard title="Card 3">...</AlmgCard></AlmgGridItem>
        </AlmgGrid>
      </div>
    </>
  );
}

Com o CSS correspondente:

@media (max-width: 768px) {
  .hidden-mobile { display: none; }
}
@media (min-width: 769px) {
  .hidden-desktop { display: none; }
}

Design Tokens para Responsividade

Os tokens CSS da biblioteca permitem manter consistência visual em qualquer tamanho de tela:

Tipografia

/* Títulos de página */
font-size: var(--font-size-almg-2xl);   /* 1.5rem — 24px */
font-size: var(--font-size-almg-xl);    /* 1.25rem — 20px */

/* Corpo */
font-size: var(--font-size-almg-base);  /* 1rem — 16px */
font-size: var(--font-size-almg-sm);    /* 0.875rem — 14px */
font-size: var(--font-size-almg-xs);    /* 0.75rem — 12px */

/* Pesos */
font-weight: var(--font-weight-almg-bold);      /* 700 */
font-weight: var(--font-weight-almg-semibold);   /* 600 */
font-weight: var(--font-weight-almg-medium);     /* 500 */
font-weight: var(--font-weight-almg-normal);     /* 400 */

Espaçamento

Escala de espaçamento baseada em múltiplos de 4px:

--spacing-almg-1:  4px      /* Separação mínima */
--spacing-almg-2:  8px      /* Gap pequeno */
--spacing-almg-3:  12px     /* Gap entre campos de formulário */
--spacing-almg-4:  16px     /* Gap padrão, padding de containers */
--spacing-almg-5:  20px
--spacing-almg-6:  24px     /* Gap grande, margens de seção */
--spacing-almg-8:  32px     /* Separação entre seções */
--spacing-almg-10: 40px
--spacing-almg-12: 48px
--spacing-almg-16: 64px
--spacing-almg-20: 80px

Cores

--color-almg-brand-font-primary;      /* Texto principal */

Boas Práticas de Layout

Container principal

Limite a largura máxima do conteúdo e centralize:

<main style={{ padding: '24px', maxWidth: '1200px', margin: '0 auto' }}>
  <Outlet />
</main>

Formulários

Limite a largura de formulários simples para melhor legibilidade:

<form style={{ maxWidth: '500px', display: 'flex', flexDirection: 'column', gap: '16px' }}>
  {/* campos do formulário */}
</form>

Grupos de botões

Use flexbox com gap para agrupar ações:

<div style={{ display: 'flex', gap: '12px' }}>
  <AlmgButton intent="primary" type="submit">Salvar</AlmgButton>
  <AlmgButton intent="secondary" type="button">Cancelar</AlmgButton>
</div>

Cabeçalho de página com ação

Para títulos com botões alinhados à direita:

<div style={{
  display: 'flex',
  justifyContent: 'space-between',
  alignItems: 'center',
  margin: '24px 0 16px',
}}>
  <h1 style={{
    fontSize: 'var(--font-size-almg-xl)',
    fontWeight: 'var(--font-weight-almg-bold)',
  }}>
    Título da Página
  </h1>
  <Link to="/entidades/novo">
    <AlmgButton intent="primary">Novo Registro</AlmgButton>
  </Link>
</div>

Grid para campos de formulário lado a lado

Use o grid para colocar campos em colunas dentro de formulários:

<AlmgGrid columns={12} gap="md">
  <AlmgGridItem span={9}>
    <AlmgInput name="rua" form={form} label="Rua" required />
  </AlmgGridItem>
  <AlmgGridItem span={3}>
    <AlmgInput name="numero" form={form} label="Número" required />
  </AlmgGridItem>
</AlmgGrid>

Cards para agrupar conteúdo

Use AlmgCard para separar visualmente seções:

<AlmgCard title="Dados Pessoais">
  <AlmgInput name="nome" form={form} label="Nome" required />
  <AlmgInput name="email" form={form} label="E-mail" type="email" required />
</AlmgCard>
